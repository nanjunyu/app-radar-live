# Changelog

本文件记录 AppRadar Live 的所有重要变更。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

---
## [1.2.1] - 2026-07-08

### 修复

- **修复 Git 扫描/升级闪退**：解决 `runCommandStreaming` 中数据竞争导致的 use-after-free 崩溃。
- **修复升级卡死**：brew/npm/CLI 升级失败时自动清理残留子进程，避免锁文件导致后续升级永远失败。
- **限制 Git 仓库发现并发数**：避免同时创建过多线程造成系统资源压力。
- **优化设置描述**：「后台自动更新」明确指向 AppRadar Live 自身，避免与应用更新中心混淆。

---
## [1.2.0] - 2026-07-08

### 新增

- **应用自动更新**：启动时自动检测新版本，支持一键后台下载、Ed25519 签名校验、自动安装并重启，全程可见下载进度。
- **设置中心改版**：新增「通用」「关于」两个分页，Tab 栏居中统一胶囊样式。
- **通用设置项**：开机自启动（默认开启）、窗口关闭行为（最小化到托盘 / 每次询问 / 退出应用）、隐藏 Dock 图标、进程刷新间隔、后台自动更新与更新提醒开关。
- **关于页**：展示应用信息、作者、开源仓库、意见反馈入口，支持「检查更新」与「更新记录」。
- **更新中心优化**：改进启动命令展示、多端口解析与 cask 应用启动选项。

### 变更

- 主题色设置迁移到「通用」分页统一管理。
- 截图改用 jsDelivr CDN 加速加载，不再受 GitHub 限流影响。

### 修复

- 修复本地覆盖安装后仍启动旧版本的问题。
- 修复自动更新安装后应用未能正确重启的问题。

---
## [1.0.0] - 2026-07-04

### 新增

- 首个正式版本发布。
- 多源应用与包管理：支持 Homebrew、Node.js / npm、Git 项目、Docker 容器及第三方安装包的扫描与更新检测。
- 轻量进程监控：展示 CPU、内存、线程等核心指标，独家端口定位与搜索。
- 极简 Docker 容器管理：查看容器状态、端口映射，支持一键启停与镜像更新。
- 防骚扰智能通知：新版本提醒与 CPU / 内存过载告警，内置冷却期。
- 5 种主题配色方案，支持自由切换。
